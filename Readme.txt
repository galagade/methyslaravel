Develop an application using Laravel, to allow for user login, and managing a database of registered vehicles and their owners.
Developing a rest API  to allow the information to be accessed via a third party. (url/api/auth) sample api -- development is still in progress. Android app to be available in version 2 of the application

#TECHNICAL
The system was built using the following technologies:
- Laravel 5.2+
- Composer
- MySQL Database
The system meets the following criteria:
- All pages, excluding the login page, must not be accessible when logged out
- Server side validation for all inputs
- Database migrations
- Scripts to seed initial database with users, and any other data
